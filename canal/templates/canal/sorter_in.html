DBC Sorter

DBC Sorter was written to keep changes on pacmod
dbc files from accidentally wreaking havoc on
Pi-Innovo Simulink models.

DBC sorter will take any dbc file and sort the signals
inside the messages based on byte position.

This keeps the signals in the correct order when used
inside a Pi-Innovo CAN-Read block.
